  1. HEXO常用命令

HEXO常用命令

  1. hexo new “新的博客名”

    新建一篇标题名为“新的博客名”的博客。

 

  1. hexo g

    生成网站静态文件到默认设置的public文件夹，它是hexo generate的缩写。

     

  2. hexo d

    自动生成网站静态文件，并部署到设定的仓库，它是hexo deploy的缩写。(可以使用hexo d -g直接进行部署生成页面)

     

  3. hexo new page about

    新建一个标题为about的页面，默认链接地址：主页地址/about/ 。

     

  4. npm install hexo -g

    安装hexo。

     

  5. npm update hexo -g

    升级hexo。

     

  6. hexo s

    运行测试，浏览器地址：http://localhost:4000/
