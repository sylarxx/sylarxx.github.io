SQL

  • SQL-Structured Query Language 结构化查询语言。

  • 是关系数据库系统（RDBMS-Relationship Database Management System）的标准语言。

​

SQL语法规则

  • SQL语句以分号结尾。
  • SQL不区分大小写。

​

SQL语法使用

SQL SELECT（选择）

  • SELECT从数据库表中选择数据，返回的数据存储于结果表中，称为结果集。

​

  • 选择表中的所有字段。

​

SQL SELECT DISTINCT（去重）

  • 返回的结果集筛选掉了相同的值。

    ​

SQL WHERE（条件筛选）

  • 通过条件筛选来查询满足条件的数据。（字符串需要‘ ’）

    ​

SQL AND、OR 、NOT（与、或、非）

  • 满足所有条件。

    ​

  • 满足其中的一个条件。

    ​

  • 非此条件的数据。

​

  • 可结合使用。

    ​

SQL ORDER BY （排序）

  • 对数据进行排列：ASC升序/DESC降序。

    ​
